Exploitation Notes

  • x86 IP - Instruction Pointer, SP - Stack Pointer
  • Trampoline: jmp esp
    • Why? We don't care about actual address of buffer now
    • Payload continues after saved return address overwrite
    • IP just jumps forward to it
    • Defeats: ASLR
    • Defeated by: Non-executable stack (NX)
  • Stack Pivot: move stack pointer to fake "stack"
    • Why? The stack can be limiting: buffer size, nx, etc.
    • Used with Return Oriented Programming (ROP)
    • Example: mov esp, (register pointing to attacker controlled "stack")
    • Defeats: DEP,NX
  • Info Leak: discover information in process memory at runtime
    • Combine with surgical ret2libc: math known libc address with desired
    • Defeats (Situational): ASLR, Stack Canary
